<task type="auto">
  <name>Task 1: Define CardEffect type system</name>
  <files>packages/server/src/game/engine/effects/types.ts</files>
  <action>
  Create the CardEffect discriminated union covering ALL effect verbs needed for Act 1 board game cards. Each member has `kind` as the discriminator.

  **Effect verbs to define:**
  - `DealDamage` — `{ hits: number, amount: number, target: 'chosen' | 'all_row' | 'all' | 'random' }`
  - `GainBlock` — `{ amount: number, target: 'self' | 'any_player' | 'all_players' }`
  - `ApplyStatus` — `{ status: 'vulnerable' | 'weak' | 'strength' | 'poison', amount: number, target: 'chosen' | 'self' | 'all_row' | 'all_enemies' | 'any_player' }`
  - `GainEnergy` — `{ amount: number }`
  - `DrawCards` — `{ count: number }`
  - `DiscardCards` — `{ count: number, target: 'self' | 'random' }`
  - `Exhaust` — `{ target: 'self' | 'chosen_hand' | 'random_hand' }`
  - `Channel` — `{ orbType: 'lightning' | 'frost' | 'dark', count: number }`
  - `Evoke` — `{ count: number }`
  - `EnterStance` — `{ stance: 'calm' | 'wrath' | 'neutral' }`
  - `GainShiv` — `{ count: number }`
  - `GainMiracle` — `{ count: number }`
  - `Scry` — `{ count: number }`
  - `AddCardToDiscard` — `{ cardId: string }` (for cards that create copies)
  - `AddCardToHand` — `{ cardId: string }`
  - `UpgradeCard` — `{ target: 'random_hand' | 'chosen_hand' | 'all_hand' }`
  - `GainGold` — `{ amount: number }`
  - `LoseHp` — `{ amount: number, target: 'self' }` (direct HP loss, not damage)
  - `HealHp` — `{ amount: number, target: 'self' | 'any_player' }`
  - `Conditional` — `{ condition: string, then: CardEffect[], else?: CardEffect[] }` (for "if in Wrath" type effects)
  - `PerX` — `{ per: string, effect: CardEffect }` (for "deal X damage per poison" type effects)
  - `Unimplemented` — `{ description: string }` (stub for cards not yet mapped)

  Also define:
  - `EffectContext` type: `{ playerId: string, targetId?: string, dieResult: number, source: 'card' | 'relic' | 'potion' | 'trigger' }`
  - `PowerEffect` type for ongoing power card effects: `{ trigger: 'start_of_turn' | 'end_of_turn' | 'on_play_attack' | 'on_play_skill' | 'on_exhaust' | 'on_take_damage', effects: CardEffect[] }`

  Use TypeScript `type` (not interface) for the discriminated union so it works with exhaustive switches.
  Use `.js` extension in all imports per project convention.
  </action>

  </verify>
  <done>CardEffect discriminated union compiles with all Act 1 effect verbs, EffectContext and PowerEffect types exported</done>
