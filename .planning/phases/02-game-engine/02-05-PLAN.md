  **GREEN phase:**

  1. `startPlayerTurn(state, rng?): CombatGameState`:
     - Increment round
     - Roll die: `Math.floor(rng() * 6) + 1` (default `rng = Math.random`)
     - For each player: reset energy to 3, block to 0, endedTurn to false
     - For each player: draw 5 cards (call `drawCards`)
     - Collect and process START_OF_TURN triggers
     - Set phase to 'PLAYER_ACTIONS'

  2. `signalEndTurn(state, playerId): CombatGameState`:
     - Set player's endedTurn to true
     - Check if ALL players have endedTurn === true
     - If yes: call `endPlayerTurn`
     - If no: return state with just the flag

  3. `endPlayerTurn(state, cardLookup): CombatGameState`:
     - Collect and process END_OF_TURN triggers
     - Apply poison tick (`applyPoisonTick`)
     - For each player: call `discardHand` (with cardLookup for retain/ethereal)
     - Set phase to 'ENEMY_TURN'

  4. `playCard(state, playerId, cardId, targetId?, cardLookup): CombatGameState`:
     - Find card in player's hand. If not found, return state unchanged.
     - Look up card cost from cardLookup. If player.energy < cost, return state unchanged (or throw? prefer return unchanged with log entry).
     - Remove card from hand, set beingPlayed = cardId (card is NOT in hand or discard during resolution)
     - Deduct energy
     - Get effects from effect registry: `getCardEffects(cardId)`
     - Build EffectContext: { playerId, targetId, dieResult: state.dieResult, source: 'card' }
     - Call `resolveCardEffects(state, effects, context)`
     - After resolution: move card based on type (lookup from cardLookup):
       - Attack/Skill: move to discardPile (unless exhaust flag → exhaustPile)
       - Power: neither discard nor exhaust (powers are in play — for Phase 2, just remove from beingPlayed)
       - If card has exhaust flag: move to exhaustPile
     - Set beingPlayed = null
     - Add combat log entry
