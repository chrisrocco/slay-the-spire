   - Import `WebSocketServer` from `ws`, `ClientMessageSchema` from shared, `RoomManager` from rooms

   - Maintain `Map<WebSocket, { roomCode: string; playerId: string }>` for connection-to-room mapping
   - `wss.on('connection', (ws) => { ... })`:
     - Set up `ws.on('message', ...)` handler:
       - Parse raw message as JSON, validate with `ClientMessageSchema.safeParse()`
       - On parse failure: send ERROR with code `INVALID_MESSAGE`
