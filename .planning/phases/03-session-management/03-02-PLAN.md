<task type="auto">
  <name>Task 1: Implement lobby handler functions</name>
  <files>packages/server/src/lobby/lobbyHandlers.ts</files>
  <action>
Create `lobbyHandlers.ts` with these exported functions. Each takes a `Room` and `playerId` plus message-specific data, and returns `{ lobby: LobbyState } | { error: string }`:

1. **selectCharacter(room, playerId, character: Character):**
   - Validate room.lobby.started is false (error: "Game already started")
   - Validate character is one of 'ironclad', 'silent', 'defect', 'watcher' (error: "Invalid character")
   - Check no other player has selected this character (error: "Character already taken")
   - Find player in lobby.players by id, set character
   - Allow re-selecting (player can change before game starts, freeing previous pick)
   - Return updated lobby

2. **toggleRule(room, playerId, rule: 'lastStand' | 'chooseYourRelic'):**
   - Validate player is host (error: "Only host can change rules")
   - Validate room.lobby.started is false (error: "Game already started")
   - Toggle the rule: `room.lobby.optionalRules[rule] = !room.lobby.optionalRules[rule]`
   - Return updated lobby

3. **startGame(room, playerId):**
   - Validate player is host (error: "Only host can start game")
   - Validate room.lobby.started is false (error: "Game already started")
   - Validate all players have selected characters (error: "All players must select a character")
   - Validate at least 1 player (error: "No players in lobby")
   - Set room.lobby.started = true
   - Return updated lobby (game initialization happens in Plan 03)

All functions mutate room.lobby in place and return the result or error.
  </action>
