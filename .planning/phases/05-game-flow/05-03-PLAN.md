export function handleSelectNode(room: Room, playerId: string, nodeId: string, rng: () => number): void

`handleSelectNode`:

2. Validate nodeId is connected to currentNodeId (unless Wing Boots relic present)
3. Update map.currentNodeId
4. Look up node type from map.nodes
5. Dispatch to appropriate room handler based on type

- For non-combat rooms: transition gamePhase back to MAP
- For combat: transition to REWARDS phase
