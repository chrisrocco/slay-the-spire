    3. Create a helper function `checkCombatEnd(room: Room): void` that:
       - Checks if `room.gameState.phase === 'COMBAT_END'` (the combat TurnPhase, NOT gamePhase)
       - If true, determines the room type by looking up `room.gameState.map.currentNodeId` in `room.gameState.map.nodes` to find the node's `type` field
       - Maps node type to reward room type: `'encounter' → 'encounter'`, `'elite' → 'elite'`, `'boss' → 'boss'`
       - For boss rooms: sets `gamePhase: 'BOSS_REWARD'`, calls `generateRewards(state, 'boss', Math.random)` for card/gold rewards AND `generateBossRelicChoices(state, state.players.length, Math.random)` for boss relics, stores both in `rewardState` (set `bossRelicChoices` on the extended state)
       - For encounter/elite rooms: sets `gamePhase: 'REWARDS'`, calls `generateRewards(state, roomType, Math.random)` and stores in `rewardState`
       - Adds gold from rewardState to each player's gold total immediately

    4. Call `checkCombatEnd(room)` at the end of `handlePlayCard` (after `processAction` and before `broadcastState`).
    5. Call `checkCombatEnd(room)` at the end of `handleEndTurn` (after `processAction` and before `broadcastState`).

    6. Create `handleUsePotion(room: Room, playerId: string, potionId: string, targetId?: string): void` that:
       - Guards: `if (!room.gameState) return;`
       - Uses the action queue (same pattern as handlePlayCard)
       - Creates action `{ type: 'USE_POTION', playerId, potionId, targetId }`
       - Calls `processAction(room.gameState, action)` and assigns result to `room.gameState`
       - Calls `checkCombatEnd(room)` (potion could kill enemies)
       - Calls `broadcastState(room)`

    7. In the existing `handleRewardPickCard`, `handleRewardPickPotion`, `handleRewardPickRelic`, and `handleRewardSkip` handlers: after updating `room.gameState`, check `areAllRewardsChosen(room.gameState)`. If true, call `handleRewardsComplete(room)`.
